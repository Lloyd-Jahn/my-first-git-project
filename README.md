## 项目简介

这是我学习 **Git 基础操作** 的练手项目。  
主要练习了以下技能：

- Git 初始化仓库 (`git init`)  
- 添加、提交和推送文件 (`git add` / `git commit` / `git push`)  
- 分支管理 (`git branch` / `git checkout` / `git merge`)  
- `.gitignore` 文件使用  
- GitHub Fork 与 Pull Request 的理解
